# skill-study v1.0.0

## Summary

Initial public release of `skill-study`, a reusable Agent Skill for learning Agent Skills and drafting better `SKILL.md` files.

## Highlights

- Explains what Agent Skills are and when to use them
- Compares Skills with MCP, workflows, and `CLAUDE.md`
- Helps users design, refine, and trigger `SKILL.md` files more effectively
- Ships with bilingual repository documentation
- Uses a simplified single-file `SKILL.md` layout

## Included Files

- `SKILL.md`
- `README.md`
- `CHANGELOG.md`
- `LICENSE`

## Install

Project scope:

```bash
mkdir -p .trae/skills/skill-study
cp SKILL.md .trae/skills/skill-study/SKILL.md
```

User scope:

```bash
mkdir -p ~/.trae/skills/skill-study
cp SKILL.md ~/.trae/skills/skill-study/SKILL.md
```

## 中文简介

`skill-study` 的首个公开版本，适合用于学习 Agent Skills，并帮助用户编写、优化和触发更高质量的 `SKILL.md` 文件。

主要内容包括：

- 解释 Agent Skills 的核心概念与适用场景
- 比较 Skill、MCP、工作流和 `CLAUDE.md`
- 帮助设计更容易被触发的 `description`
- 提供更适合 GitHub 展示的双语文档
