# Release Title

```text
skill-study v1.0.0 - First Public Release
```

# Release Body

## English

Initial public release of `skill-study`, a reusable Agent Skill for learning Agent Skills and drafting better `SKILL.md` files.

### Highlights

- Explains what Agent Skills are and when to use them
- Compares Skills with MCP, workflows, and `CLAUDE.md`
- Helps users design, refine, and trigger `SKILL.md` files more effectively
- Includes bilingual repository documentation
- Uses a simplified single-file `SKILL.md` layout

### Included

- `SKILL.md`
- `README.md`
- `CHANGELOG.md`
- `LICENSE`

### Install

Project scope:

```bash
mkdir -p .trae/skills/skill-study
cp SKILL.md .trae/skills/skill-study/SKILL.md
```

User scope:

```bash
mkdir -p ~/.trae/skills/skill-study
cp SKILL.md ~/.trae/skills/skill-study/SKILL.md
```

## 中文

这是 `skill-study` 的首个公开版本。它是一个可复用的 Agent Skill，适合用于学习 Agent Skills，并帮助用户编写、优化和触发更高质量的 `SKILL.md` 文件。

### 本次发布亮点

- 解释 Agent Skills 的核心概念和适用场景
- 比较 Skill、MCP、工作流和 `CLAUDE.md`
- 帮助用户设计更容易被触发的 `description`
- 提供更适合 GitHub 展示的中英双语文档
- 使用更清晰的单文件 `SKILL.md` 结构

### 包含内容

- `SKILL.md`
- `README.md`
- `CHANGELOG.md`
- `LICENSE`
